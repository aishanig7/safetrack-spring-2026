Overview:
The firmware implements a LoRa-based mesh communication system where multiple nodes can transmit, receive, and forward packets until they reach a designated head node. The system is designed to support SOS communication using GPS data without relying on cellular or internet infrastructure. 

In this project, device is able to:
Reads inputs (button, GPS, radio)
Processes data (builds packets, parses packets)
Controls outputs (display, buzzer, LED)
Handles communication (LoRa transmission and reception)
